### **Core Problem We Solve**

- People don't collect waste because there's no incentive
- Government/NGOs lack real-time visibility of waste locations
- No system exists to turn public participation into measurable environmental impact

### **Our Solution**

We gamify waste collection with:

1. **Citizens** collect plastic waste and submit photos with location
2. **AI verification** processes images for plastic detection and weight estimation
3. **Municipal workers** verify and approve submissions
4. **Citizens earn points** that can be redeemed for real rewards
5. **Real-time impact tracking** shows environmental benefits

### **Camera Permissions**

```xml

<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
```

### **File Provider**

Images are shared securely using FileProvider. Configuration in `AndroidManifest.xml`:

```xml
<provider
    android:name="androidx.core.content.FileProvider"
    android:authorities="${applicationId}.fileprovider"
    android:exported="false"
    android:grantUriPermissions="true">
    <meta-data
        android:name="android.support.FILE_PROVIDER_PATHS"
        android:resource="@xml/file_paths" />
</provider>
```

## ğŸš€ Publishing to Play Store

### **Step 1: Generate Signing Key**

```bash
keytool -genkey -v -keystore trash2cash-release-key.jks -keyalg RSA -keysize 2048 -validity 10000 -alias trash2cash
```

### **Step 2: Configure build.gradle.kts**

Add to `app/build.gradle.kts`:

```kotlin
android {
    signingConfigs {
        create("release") {
            storeFile = file("trash2cash-release-key.jks")
            storePassword = "YOUR_STORE_PASSWORD"
            keyAlias = "trash2cash"
            keyPassword = "YOUR_KEY_PASSWORD"
        }
    }
    
    buildTypes {
        release {
            signingConfig = signingConfigs.getByName("release")
            isMinifyEnabled = true
            proguardFiles(
                getDefaultProguardFile("proguard-android-optimize.txt"),
                "proguard-rules.pro"
            )
        }
    }
}
```

### **Step 3: Build Release APK/AAB**

```bash
# Build release APK
./gradlew assembleRelease

# Build Android App Bundle (for Play Store)
./gradlew bundleRelease
```

Output location:

- APK: `app/build/outputs/apk/release/app-release.apk`
- AAB: `app/build/outputs/bundle/release/app-release.aab`
